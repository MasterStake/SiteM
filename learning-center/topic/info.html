How do I backup my wallet.dat file?

algorithm SHA-256
source branch 0.21

updated October 28, 2022, 3:31 pm

Backup your wallet.dat file with the following instructions.

Close your wallet.

Open "Run" with the keyboard combination winkey + r.

Enter the following text behind "Open": explorer

Press on the button "OK".

Click on the menu bar, type the following text %appdata% and press on the enter key. enter

Open the blockchain directory of your coin, the directory name is the name of your coin.

Open the directory wallets and copy the file wallet.dat to a safe location.
